Uma breve explicação de Web Designer

O web desing é uma extenção da prática do desing gráfico,onde o foco do projeto é a ciração de web sites e documentos disponíveis no ambiente da Word Wide Web.

é responsável estético de um site quanto por seu projeto funcional.Ou seja,o web desingner se preocupa com a aparência e com a funcionalidade de um website,pensando na navegabilidade na interação que os usuários terão com a página da internet criada.
